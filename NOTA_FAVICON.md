# Nota sobre el Favicon

## Tamaño del Favicon en la Pestaña

El tamaño visual del favicon en la pestaña del navegador está **limitado por el navegador mismo**:
- Chrome/Edge: 16x16 píxeles
- Firefox: 16x16 píxeles  
- Safari: 16x16 o 32x32 píxeles (dependiendo del zoom)

## Solución: Archivo de Alta Resolución

Para que el favicon se vea **más nítido y claro**, el archivo debe ser de **alta resolución**:

### Tamaños Recomendados:
- **Mínimo**: 32x32 píxeles
- **Recomendado**: 64x64 píxeles
- **Óptimo**: 128x128 o 256x256 píxeles

### Formato:
- **PNG**: Mejor calidad, soporta transparencia
- **ICO**: Formato tradicional, puede contener múltiples tamaños en un solo archivo
- **SVG**: Se escala perfectamente (si el navegador lo soporta)

## Cómo Mejorar el Favicon Actual

1. **Asegúrate de que `icono-pestana.png` sea de alta resolución**:
   - Abre el archivo en un editor de imágenes
   - Verifica que tenga al menos 64x64 píxeles (preferiblemente 128x128)
   - Si es más pequeño, redimensiona a 128x128 o 256x256

2. **Optimiza el diseño**:
   - Usa colores contrastantes
   - Evita detalles muy pequeños que se pierdan en 16x16
   - Diseña pensando en que se verá muy pequeño

3. **Crea un favicon.ico** (opcional):
   - Puedes crear un archivo `.ico` que contenga múltiples tamaños
   - Herramientas online: https://favicon.io/favicon-converter/

## Nota Importante

**El tamaño visual NO puede ser el doble** porque está limitado por el navegador. Lo que SÍ podemos hacer es:
- ✅ Usar un archivo de alta resolución para mejor nitidez
- ✅ Optimizar el diseño para que se vea claro en tamaño pequeño
- ✅ Agregar múltiples tamaños para diferentes dispositivos

El navegador siempre mostrará el favicon en su tamaño estándar (16x16 o 32x32), pero con un archivo de alta resolución se verá mucho más nítido.
